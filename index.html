配置 API 密钥
AI 仅使用 Google Gemini。密钥保存在浏览器本地（localStorage），不会上传服务器。
如何获取 Gemini API Key：
1
访问 aistudio.google.com/apikey，使用 Google 账号登录
2
点击「Create API key」→ 选择项目（新建或已有）→ 复制 Key
3
Key 格式：AIza 开头，约 39 字符。请求会按顺序尝试多个 Gemini 模型（lite / 1.5 / 2.0），某一模型配额用尽时自动换下一个；若全部失败请在 AI Studio 查看配额与结算
可选 · Supabase 云数据库
配置 Supabase 后数据将同步到云端（可选）：
1
访问 supabase.com → 创建免费项目
2
Project Settings → API → 复制「Project URL」和「anon public」Key
3
在 SQL Editor 执行建表语句：
create table keywords (id serial primary key, kw text, type text, status text, article text, position text, priority text, note text);
create table articles (id serial primary key, title text, keywords text[], status text);
